Laboratorio de comandos

Cuenta archivos, carpetas y uso de disco

Aprende a medir elementos del sistema con comandos reales, ejemplos y scripts listos para ejecutar.

Comando destacado

$ ls -1 | wc -l

Cuenta archivos del directorio actual.

Objetivo del laboratorio

Comprender como contar archivos, carpetas y medir el uso de disco con herramientas basicas de Linux.

Ejemplo: find, du, tree

Comandos clave

Presiona Ejecutar para ver la salida simulada.

ls | wc -l

Cuenta los elementos listados.

$ ls -1 | wc -l

find

Busca archivos por tipo o patron.

$ find . -type f | wc -l

find (dirs)

Cuenta directorios dentro de una ruta.

$ find . -type d | wc -l

du

Mide uso de disco por carpeta.

$ du -sh .

df

Espacio libre por particion.

$ df -h

tree

Visualiza estructura de carpetas.

$ tree -L 2
