# Analizador del sistema Linux

Laboratorio de comandos para contar archivos, carpetas y uso de disco.

---

## Descripcion

Proyecto educativo que muestra tecnicas para medir elementos del sistema con comandos Linux.

### Que hace este proyecto?

- **Conteo rapido**: `ls | wc -l` para listar elementos
- **Busqueda precisa**: `find` por tipo y profundidad
- **Uso de disco**: `du` y `df` para tamanos y particiones
- **Visualizacion**: diagramas de carpetas con `tree`
- **Scripts bash**: ejemplos listos para ejecutar

---

## Instalacion y uso

### Requisitos

- Navegador moderno
- Linux o WSL para scripts bash

### Uso

```bash
# Abrir el sitio
start index.html

# Ejecutar scripts
bash bash/count_files.sh
bash bash/count_dirs.sh
bash bash/disk_usage.sh
```
